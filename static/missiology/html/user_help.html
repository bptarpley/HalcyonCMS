User Guide

Creating Normal Users

Visit the registration page and fill out the appropriate values.

Creating Bloggers

  1. Create a normal user, copy their username.
  2. While logged in as an admin, scroll to the bottom of a page, and in the text field next to the "Manage User" button, paste their name in and click the button.
  3. Check the "Can blog?" checkbox and click "Save."

Creating Bibliographers

  1. Create a blogger.
  2. While logged in as an admin, go the the user administration page and click on the blogger's username.
  3. Scroll down to the "User permissions" pane, and under "Available user permissions," select every entry that begins with the word "bibliography." Click the blue right arrow to move those selections into the "Chosen user permissions." Click "Save."

Creating Admins

  1. While logged in as an admin, paste a username next to the "Manage User" button on in the Admin Tools section of a page and click the button.
  2. Check the "Is Admin?" box and click "Save."
